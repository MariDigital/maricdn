== Description ==

Speed up your website with **MariCDN by MariHost** Content Delivery Network. This plugin allows you to easily enable MariCDN on your WordPress and enjoy greatly improved loading times around the world. Even better, it takes just a minute to set up, just enter the name of your Pull Zone and youâ€™re all set.

= How does it work? =
The plugin will automatically configure your website to replace existing static content links with the CDN links. This will greatly speed up your content.

= Features =
* Replace static links with CDN links
* Automatic HTTPS configuration 
* Include or exclude specific directories or phrases
* Set a custom CDN hostname

= System Requirements =
* PHP >=7.0
* WordPress >=5.0

== Installation ==
 
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
